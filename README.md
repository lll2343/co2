# 项目开发文档

> 运行方式、各部分内容概括

## 本地运行方式

1. 进入本项目的[github链接](https://github.com/lll2343/co2)
2. 打开命令行终端，将本项目使用git clone至本地
3. 进入项目文件夹，运行 npm i 安装必要组件
4. 安装完成后，运行 npm run dev，进入生成的网址即可
